# Markdown 学习笔记

> **Markdown** 是一种轻量级标记语言，它使用纯文本格式编写内容，并通过简单的标记语法来实现格式化效果。

## 基本语法

**粗体**使用 `**` 或 `__`  
*斜体* 使用 `*` 或 `_`  
~~删除线~~ 使用 `~~` 包裹文字
